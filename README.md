
# Alumni Portal

The Alumni Portal is a web-based platform designed to connect alumni with their university and with each other. It provides a centralized space for alumni to update their profiles, share professional history, participate in events, view notices, and engage with the alumni community. Administrators can manage alumni information, post announcements, track event responses, and maintain a reliable database.
##  Running the Project

### 1. Install Go
Ensure Go (1.20+) is installed:

```bash
go version
```
### 2. Clone Repository
```bash
git clone https://github.com/dbms-project-2025-code/dbms-project-3
```

### 3. Run Web App
```bash
go run .
```
##  Tech Stack

- Go 1.21
- SQLite database
- HTML, CSS, JavaScript
